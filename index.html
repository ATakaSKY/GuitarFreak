Types

Acoustic

Has gentle rhythmic chordal strumming. Suitable for a beginner.

Electric

Solid-bodied guitars designed to be plugged into an amplifier.

Twelve-string

Has six regular strings and a second set of thinner strings.

Steel

Originated from Hawaiin. Played horizontally across the player's lap.

Bass

Has a long neck and thick strings. The standard bass has four strings though five and six string basses are available which extends the range of the instrument.

Double Neck

Designed so that two guitar necks can share one body. This design allows the guitarist to switch between either neck with ease.
